# Pre-React-Native Transition Archive

This folder preserves the former active frontend planning docs from the period when ATTREQ was being positioned and planned as a Next.js PWA.

## Why These Docs Were Archived

The project direction changed:

- the repo still contains `apps/web`,
- but React Native became the new primary frontend direction,
- so the old PWA frontend planning docs are no longer authoritative.

These files are kept so earlier assumptions, task breakdowns, and historical planning decisions are not lost.

- Archive docs do not override active docs.
- If archive docs conflict with current code or current-status docs, the archive is wrong by definition.
